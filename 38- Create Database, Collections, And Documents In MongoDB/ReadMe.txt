Working with cmd = database

1) to open mongo shell type mongo in cmd
2) to check database type show dbs
3) to create a document in database type use DocumentName if the DocumentName not exist it will automatically create that
4) to check the current document type db
5) to check collections of your document type show collections
6) to check  data of your collection type db.DocumentName.find().pretty()
7) to add single data in current collection type db.CollectionName.insertOne({name: "Aamir", subject: "mongoDb"})
