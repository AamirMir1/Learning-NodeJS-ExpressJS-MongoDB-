Working with cmd = database

first open your cmd

if you want to run mongodb shell then open copy your path of mongod
e.g  C:\Program Files\MongoDB\Server\5.0\bin\mongo.exe         paste this in a string in your cmd

if you want to run mongodb database  then open copy your path of mongo
e.g C:\Program Files\MongoDB\Server\5.0\bin\mongod.exe      paste this in a string in your cmd

add a environment variable for short hand
e.g    C:\Program Files\MongoDB\Server\5.0\bin

mongo =  C:\Program Files\MongoDB\Server\5.0\bin\mongo.exe
mongod = C:\Program Files\MongoDB\Server\5.0\bin\mongo.exe

to check the version of your mongodb, type --version after code
e.g   mongo --version or  "C:\Program Files\MongoDB\Server\5.0\bin\mongo.exe" --version
